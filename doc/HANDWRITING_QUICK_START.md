# 手写输入功能快速上手指南

## 🎯 快速开始

### 在哪里使用？

在学生端作业答题页面，当遇到**开放式问题（Open-Ended Questions）**时，你会看到三个输入模式选项。

### 三种输入模式

```
┌─────────────────────────────────────────────┐
│  [Type In]  [Write]  [Upload]              │
└─────────────────────────────────────────────┘
```

## 📝 模式 1: Type In（打字输入）

**适用场景**: 纯文字答案

**操作方法**:
1. 点击 "Type In" 按钮
2. 在文本框中直接输入答案
3. 系统会自动保存

## ✏️ 模式 2: Write（手写输入）

**适用场景**: 数学公式、图表、草稿演算

**操作方法**:
1. 点击 "Write" 按钮
2. 使用工具栏：
   ```
   画笔大小: [●] [●●] [●●●] [●●●●]
   颜色: [黑] [红] [蓝] [绿]
   操作: [撤销] [清除] [保存绘图]
   ```
3. 在白色画布上直接书写
4. 完成后点击"Save Drawing"保存

**iPad 使用技巧**:
- ✅ 使用 Apple Pencil 效果最佳
- ✅ 可以用手指绘画
- ✅ 支持手掌防误触
- ✅ 流畅低延迟

## 📸 模式 3: Upload（上传扫描）

**适用场景**: 纸质作业拍照、已完成的手写作业

**操作方法**:
1. 点击 "Upload" 按钮
2. 点击 "Take Photo / Upload Image"
3. 选择照片或拍照
4. 支持同时上传多张图片

## 🎨 画笔工具详解

### 画笔大小
- **1px**: 细线，适合写小字
- **2px**: 常规，推荐日常使用
- **4px**: 粗线，适合标注重点
- **6px**: 最粗，适合画图

### 颜色选择
- **黑色**: 默认颜色，正常书写
- **红色**: 标注重点、错题订正
- **蓝色**: 辅助说明、标记
- **绿色**: 补充信息

### 操作按钮
- **Undo（撤销）**: 后悔了？撤销上一步
- **Clear（清除）**: 全部清空，重新开始
- **Save Drawing（保存绘图）**: 将画布保存为图片

## 💡 使用技巧

### ✅ 推荐做法

1. **数学题**: 使用 Write 模式，选择细画笔（1-2px），黑色书写
2. **需要标注**: 先用黑色写答案，再用红色标注重点
3. **画图题**: 使用 Write 模式，可以用不同颜色区分
4. **长文本**: 使用 Type In 模式，方便修改
5. **纸质作业**: 使用 Upload 模式，确保照片清晰

### ❌ 避免的做法

1. ❌ 画布内容未保存就切换模式（会丢失）
2. ❌ 上传模糊的照片
3. ❌ 使用过大的画笔写小字
4. ❌ 忘记点击"Save Drawing"

## 📱 不同设备的使用建议

```

推荐使用 Write 模式，就像在纸上写字一样
```

### 手机
```

先在纸上写，然后拍照上传
```

### 电脑 + 鼠标
```
💻 建议使用 Type In 模式
鼠标不适合手写，打字更方便
```

### 平板 + 手指
```
👆 可以使用 Write 模式
选择较大的画笔（4-6px）
```

## 🔧 常见问题

### Q: 切换模式后，之前的内容会丢失吗？
A: 
- Type In 的文字会保留
- Write 模式需要先点"Save Drawing"才能保存
- Upload 的图片会一直保留

### Q: 可以同时使用多种模式吗？
A: 可以！你可以先打字，再添加手写图片，还能上传照片。所有内容都会保存。

### Q: 手写的内容保存在哪里？
A: 点击"Save Drawing"后，手写内容会转换为图片，显示在"Uploaded Content"区域。

### Q: 可以删除已保存的图片吗？
A: 可以！在图片预览上点击右上角的 ❌ 按钮即可删除。

### Q: 画布支持橡皮擦吗？
A: 暂不支持，但可以使用"Undo"撤销，或"Clear"清空重写。

### Q: iPad 上手掌会触发绘画吗？
A: 不会！我们使用了防误触技术，手掌不会触发绘画。

## 🎓 实例演示

### 示例 1: 数学计算题

```
1. 点击 "Write"
2. 选择画笔大小 2px，黑色
3. 写出计算过程
4. 用红色标注答案

```

```
1. 点击 "Type In"
2. 输入文字答案
3. 切换到 "Write"
4. 画图表说明
5. 点击 "Save Drawing"
6. 完成！（文字和图片都已保存）
```

### 示例 3: 纸质作业

```
1. 在纸上完成作业
2. 点击 "Upload"
3. 拍照上传（可多张）
4. 检查照片清晰度
5. 提交作业
```
